#  CRUD Productos - Angular + .NET

Proyecto Full Stack desarrollado con Angular 17 y .NET 8 Web API conectado a SQL Server.

##  Autor
Diego Cabrera

---

##  Tecnologías
- Angular 17
- .NET 8 Web API
- Entity Framework Core
- SQL Server
- Bootstrap 5

---

##  Funcionalidades
-  Listar productos
-  Crear productos
-  Editar productos
-  Eliminar productos
-  Conexión Angular + .NET

---

##  Estructura
Corre en:
http://localhost:5028

Swagger:
http://localhost:5028/swagger

Ejecutar:
ng serve

Abrir en navegador:
http://localhost:4200
